Identifique y corrija los errores en cada uno de los siguientes segmentos de código:

a) while ( c <= 5 ) { producto *= c; ++c;

a) Error. Falta la llave derecha de cierre del cuerpo de la instrucción while.

b) if ( genero == 1 ) document.writeln( "Mujer" ); else; document.writeln( "Hombre" );

b) Error. El ; después de else produce un error lógico. La segunda instrucción de salida siempre se ejecutará.
