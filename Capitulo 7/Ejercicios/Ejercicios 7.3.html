Ejercicios de autoevaluación.

Escriba instrucciones en JavaScript para realizar cada una de las siguientes tareas:

a) Asignar la suma de x y y a z, e incrementar el valor de x en 1 después del cálculo. Usar sólo una instrucción.

a) z = x++ + y;

b) Evaluar si el valor de la variable cuenta es mayor que 10. De ser así, imprimir "Cuenta es mayor que 10".

b) if ( cuenta > 10 ) document.writeln( "Cuenta es mayor que 10" );

c) Decrementar la variable x en 1, luego restarla a la variable total. Usar sólo una instrucción.

c) total -= --x;

d) Calcular el residuo después de dividir q entre divisor y asignar el resultado a q. Escriba esta instrucción de dos maneras distintas.

d) q %= divisor; q = q % divisor;
