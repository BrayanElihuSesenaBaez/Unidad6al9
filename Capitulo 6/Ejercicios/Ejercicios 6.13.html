Dada y = ax^3 + 7, ¿cuáles de las siguientes intrucciones de JavaScript son correctas para esta ecuación?

a) y = a * x * x * x + 7;

La siguiente ecuación es correcta.

b) y = a * x * x * (x + 7);

La siguiente ecuación es incorrecta.

c) y = (a * x) * x * (x + 7);

La siguiente ecuación es incorrecta.

d) y = (a * x) * x * x + 7;

La siguiente ecuación es correcta.

e) y = a* (x * x * x ) + 7;

La siguiente ecuación es correcta.

f) y = a * x * (x * x + 7);

La siguiente ecuación es incorrecta.
